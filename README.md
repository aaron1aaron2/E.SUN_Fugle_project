## 討論內容
### 想法
* 藉由投資人的輪廓跟公司性質來進行推薦
* 藉由公司的產業前景與各種指標來判斷是否要推薦此公司

  投資人的輪廓：
  * 風險承受度
  * 投資經驗
  * 對金融市場的了解程度
  * 資金多寡
  * 可以參考風險量表（基金開戶會填，股票開戶不一定有填），但可以參考量表的問題，從開戶資料裏自己建構

### 結論
以股價為主，基本面、大盤面為輔，附加新聞資訊等，來推播通知。<br>
因為以價格為主要推播因素，所以投顧的建議買賣價格及5、20、120MA（價位、方向、排列)，應該要作主要考量<br>
另外也可以參考股票淨值與本益比來判斷。

### 問題
* 要以何種方式提供資訊？
* 推播通知的時間點？
    -> e.g. 股價大漲大跌的時候，投資人可能會想看新聞
* 新聞有授權問題，只能爬標題，想看新聞內容需連結到原新聞網頁
